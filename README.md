# Ethical Hacking Basic

Chhota sa educational repo for learning how common security concepts work in a safe, authorized lab environment.

## What's inside
- `increption.py`: RC4 encryption/decryption toy example (string demo).
- `keylogger.py`: Keyboard event logging demo using `pynput`.
- `keylog.txt`: Output file generated by the keylogger.

## Requirements
- Python 3.8+
- `pynput` (only needed for `keylogger.py`)

## Setup
```bash
pip install pynput
```

## Usage (authorized/testing only)
```bash
python increption.py
python keylogger.py
```

## Ethics & Legal
- Use only on systems you own or have explicit permission to test.
- Never run on real users without written consent.
- This repo is for learning and awareness only.

## Notes
- RC4 is outdated and not secure for real-world use.
- Keylogging is a sensitive topic; keep it strictly for legal labs and demos.
